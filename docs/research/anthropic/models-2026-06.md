Grounded in web research (per the research-loop discipline), cross-checked against
the session's authoritative model IDs. Current as of June 2026.

## Models, IDs, and per-million-token pricing

| Model | API id | Input $/Mtok | Output $/Mtok | Tier role |
|---|---|---|---|---|
| Opus 4.8 (flagship, rel. 2026-05-28) | `claude-opus-4-8` | 5.00 | 25.00 | deepest cognition |
| Sonnet 4.6 | `claude-sonnet-4-6` | 3.00 | 15.00 | mid / balanced |
| Haiku 4.5 | `claude-haiku-4-5-20251001` (alias `claude-haiku-4-5`) | 1.00 | 5.00 | fast / cheap |
| Fable 5 | `claude-fable-5` | — | — | FORTUNA's prior synthesis default |

- Output costs **5×** input across all models.
- Batch processing **50% cheaper**; prompt caching cuts **cached input 90%**.
- Opus 4.8 output ($25) is **5× Haiku** ($5) — budget impact is large for the
  deep tier.

## FORTUNA mapping (spec 5.9 cognition tiering — TWO tiers today)

- `[cognition].synthesis_model` (deep belief pricing) → **`claude-opus-4-8`**
- `[cognition].triage_model` (cheap trigger gating) → **Haiku 4.5**
- **No mid-tier slot exists today** — FORTUNA cognition is synthesis + triage
  only. Sonnet 4.6 has no home until a mid-tier cognition role is added (e.g. a
  shadow/comparator model or a triage-escalation tier). Tracked, not wired.
